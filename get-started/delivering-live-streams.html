라이브 스트림 제공

이 항목에서는 Brightcove가 라이브 스트림을 제공하기 위해 제공하는 옵션에 대해 설명합니다.

Video Cloud 라이브 스트리밍 비디오 기능을 사용하면 모든 장치 또는 브라우저에 고품질 라이브 스트리밍 브로드 캐스트를 제공 할 수 있습니다. 사용자 지정 개발이나 복잡한 통합없이 라이브 스트림을 제공 할 수 있지만 사용자 지정 요구 사항이 있거나 사용자 지정 통합이 필요한 경우 API를 사용할 수 있습니다.

참고: 라이브 스트리밍 지원은 Video Cloud 엔터프라이즈 게시자에게만 제공됩니다.

Video Cloud는 DVR을 통한 라이브 스트리밍도 지원합니다. 브라이트코브 플레이어가 DVRUX플러그인으로 구성되면 라이브 스트림은 시청자의 디지털 비디오 레코더를 통해 재생되는 것처럼 작동합니다.

Brightcove는 라이브 스트림을 제공하기위한 여러 옵션을 제공합니다. 이 항목에는 다음 섹션이 포함되어 있습니다.

  • 하드웨어 대 클라우드 기반 인코딩
  • 소프트웨어 기반 인코더를 사용한 간단한 라이브 스트림
  • Video Cloud Live 모듈을 사용한 라이브 스트리밍
  • 하드웨어 기반 인코더를 사용한 라이브 스트리밍

하드웨어 대 클라우드 기반 인코딩

Brightcove는 하드웨어 및 클라우드 기반 인코딩을 모두 지원합니다. 다양한 요인이 하드웨어 또는 클라우드 기반 인코딩이 이벤트에 가장 적합한 지 여부를 결정할 수 있습니다.

하드웨어 인코딩을 사용하는 경우

  • 용량이 알려진 경우 24x7 선형 피드
  • 단일 위치에 기반한 아키텍처
  • 방송 형식 지원

클라우드 인코딩을 사용하는 경우

  • 이벤트
  • 용량 요구 사항을 알 수없는 경우
  • 분산 된 조직의 경우
  • 이벤트 타이밍을 알 수없고 속도가 중요한 경우
  • 대역폭 병목 현상이있는 경우

소프트웨어 기반 인코더를 사용한 간단한 라이브 스트림

이 옵션을 사용하면 Akamai Stream Packaging 및 라이브 이벤트를 Brightcove Player로 스트리밍하기위한 소프트웨어 기반 인코더. 자세한 내용은 Akamai 스트림 패키징 및 텔레스 스트림 와이어캐스트를 사용하여 라이브 스트림생성 단원을 참조하십시오 .

Video Cloud Live 모듈을 사용한 라이브 스트리밍

Video Cloud Live 모듈은 데스크톱 및 모바일 장치 모두에 라이브 이벤트를 브로드 캐스트하는 데 도움이되는 사용하기 쉬운 인터페이스를 제공합니다. 프로그래밍이 필요하지 않습니다. 라이브 모듈에 이벤트 정보를 입력하면 현장 인코더에 연결하는 일련의 인코더 설정이 제공됩니다. 모든 트랜스 코딩은 클라우드에서 수행되므로 하드웨어 기반 트랜스 코더가 필요하지 않습니다. 자세한 내용은 다음 문서 중 하나를 참조하십시오.

  • Live 모듈을 사용하여 라이브 이벤트 생성 및 관리 -Video Cloud Live 모듈 사용에 대한 모든 세부 정보 제공
  • 단계별: Telestream Wirecast와 함께 라이브 모듈 사용 -Telestream Wirecast와 함께 Video Cloud Live 모듈 사용에 대한 단계별 가이드 제공
  • 단계별: 오픈 브로드 캐스터 소프트웨어 (OBS)와 함께 라이브 모듈 사용 -Open Broadcaster 소프트웨어와 함께 Video Cloud Live 모듈 사용에 대한 단계별 가이드 제공
  • 라이브 모듈 지침 및 모범 사례 -고품질의 안정적인 라이브 스트리밍 경험을 보장하는 데 도움이되는 일련의 모범 사례 및 권장 사항을 제공합니다.

라이브 API

Live API는 라이브 스트리밍 이벤트를 만들고 관리 할 수있는 REST 기반 API입니다. 옵션 기능은 다음과 같습니다.

  • 서버 측 광고 삽입 (SSAI)
  • AES-128 암호화
  • 라이브 스트림에서 가져온 클립에서 주문형 비디오 자산 만들기
  • DVR 기능
  • 여러 CDN

자세한 내용은 Live API 설명서를 참조하십시오 .

하드웨어 기반 인코더를 사용한 라이브 스트리밍

이 옵션은 하드웨어 기반 인코더를 사용하여 라이브 스트림을 인코딩하고 CDN에 제공합니다. Video Cloud 측에서는 스트림을 호스팅 할 원격 자산을 만든 다음 라이브 스트림 용 플레이어를 게시 할 수 있습니다.
