페이지 내용

    라이브 스트림 전송

    이 항목에서는 라이브 스트림을 전송하기 위해 브라이트코브가 제공하는 옵션에 대해 설명 합니다.

    Video Cloud 라이브 스트리밍 비디오 기능을 사용하면 모든 장치 또는 브라우저로 고품질 라이브 스트리밍 방송을 전송할 수 있습니다. 사용자 지정 개발이나 복잡한 통합 없이 라이브 스트림을 제공할 수 있지만, 사용자 지정 요구 사항이 있거나 사용자 지정 통합이 필요한 경우에는 API를 사용할 수 있습니다.

    참고: 라이브 스트리밍 지원은 Video Cloud Pro 및 Enterprise 게시자만 이용할 수 있습니다.

    Video Cloud는 DVR이 포함된 라이브 스트리밍도 지원합니다. Brightcove Player에 DVRUX 플러그인이 설정되어 있을때, 라이브 스트림은 시청자의 디지털 비디오 레코더를 통해 재생되는 것처럼 작동합니다.

    브라이트코브는 라이브 스트림을 전송할 수 있는 몇 가지 옵션을 제공합니다. 이 항목에는 다음과 같은 섹션이 포함되어 있습니다.

    • 하드웨어 대 클라우드 기반 인코딩
    • 소프트웨어 기반 인코더를 사용한 간단한 라이브 스트림
    • Video Cloud Live 모듈을 사용한 라이브 스트리밍
    • 하드웨어 기반 인코더를 사용한 라이브 스트리밍

    하드웨어 대 클라우드 기반 인코딩

    브라이트코브는 하드웨어와 클라우드 기반 인코딩을 모두 지원합니다. 다양한 요인에 따라 하드웨어 또는 클라우드 기반 인코딩이 이벤트에 가장 적합한지 결정할 수 있습니다.

    하드웨어 인코딩을 사용하는 경우

    • 용량이 알려진 24x7 선형 피드
    • 단일 위치를 기반으로 하는 아키텍처
    • 브로드캐스트 형식 지원

    클라우드 인코딩을 사용하는 경우

    • 이벤트 기반
    • 용량 요구 사항을 알 수 없는 경우
    • 분산된 조직의 경우
    • 이벤트 타이밍을 알 수 없으며 속도가 중요한 경우
    • 대역폭 병목 현상이 있는 경우

    소프트웨어 기반 인코더를 사용한 간단한 라이브 스트림

    이 옵션을 통해 Akamai Stream Packaging 및  소프트웨어 기반 인코더를 사용하여 라이브 이벤트를 Brightcove Player로 스트리밍할 수 있게 합니다. 자세한 내용은 Akamai 스트림 패키징 및 Telestream Wirecast를 사용하여 라이브 스트림 생성을 참조하십시오.

    비디오 클라우드 라이브 모듈을 사용한 라이브 스트리밍

    Video Cloud Live 모듈은 사용하기 쉬운 인터페이스를 제공하여 데스크톱 및 모바일 장치 모두에 라이브 이벤트를 방송하는데 도움이 됩니다. 프로그래밍이 필요하지 않습니다. 라이브 모듈에 이벤트 정보를 입력하면 현장 인코더에 연결하는 일련의 인코더 설정이 제공됩니다. 모든 트랜스코딩은 클라우드에서 수행되므로 하드웨어 기반 트랜스코더가 필요하지 않습니다. 자세한 내용은 다음 문서 중 하나를 참조하십시오.

    • Live 모듈을 사용하여 라이브 이벤트 생성 및 관리 -Video Cloud Live 모듈 사용에 대한 모든 세부 정보 제공
    • 단계별: Telestream Wirecast와 함께 라이브 모듈 사용 -Telestream Wirecast와 함께 Video Cloud Live 모듈 사용에 대한 단계별 가이드 제공
    • 단계별: Open Broadcaster Software (OBS)와 함께 라이브 모듈 사용 -Open Broadcaster Software 함께 Video Cloud Live 모듈 사용에 대한 단계별 가이드 제공
    • 라이브 모듈 지침 및 모범 사례 -고품질의 안정적인 라이브 스트리밍 경험을 보장하는 데 도움이되는 일련의 모범 사례 및 권장 사항을 제공합니다.

    라이브 API

    라이브 API는 라이브 스트리밍 이벤트를 만들고 관리할 수 있는 REST 기반 API입니다. 옵션 기능은 다음과 같습니다.

    • 서버 측 광고 삽입 (SSAI)
    • AES-128 암호화
    • 라이브 스트림에서 가져온 클립에서 주문형 비디오 에셋 만들기
    • DVR 기능
    • 여러 개의 CDN

    자세한 내용은라이브 API 문서를 참조하십시오.

    하드웨어 기반 인코더를 사용한 라이브 스트리밍

    이 옵션은 하드웨어 기반 인코더를 사용하여 라이브 스트림을 인코딩하고 CDN에 전달합니다. Video Cloud 에서 스트림을 호스팅할 원격 에셋 (remote asset)을 만든 다음 라이브 스트림용 플레이어를 게시할 수 있습니다.
