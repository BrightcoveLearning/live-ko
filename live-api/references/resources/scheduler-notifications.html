스케줄러 알림

스케줄러로 워크 플로를 만들 때 선택적으로 알림을 구성 할 수 있습니다. 작업을 완료한 후 스케줄러는 http 메소드 POST를 사용하여 구성된 엔드포인트에 어떤 작업이 수행되었는지 및 성공/실패에 대한 세부 정보와함께 알림을 보냅니다. 예정된 작업에 대한 사전 알림을 구성 할 수도 있습니다. 를 참조하십시오 API 참조워크플로 유형에 대한 알림을 구성합니다.

알림 본문

서버가알림_url에 알림을 보내면 요청에 JSON 본문이 포함됩니다.

들 기술
동작 액션 이름. 동작 보기
워크 플로우 작업 완료 후 업데이트 된 워크 플로. 구조는 워크 플로 유형에 따라 다릅니다.
오류 작업을 완료하는 동안 오류가 발생하면이 필드에 오류가 포함됩니다. 값은 작업 유형에 따라 다릅니다.
