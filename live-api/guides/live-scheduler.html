소개

라이브 스케줄러를 사용하면정적 진입점 (SEP) 작업에 대한 작업을 예약할 수 있습니다. . 시간으로 스케줄러 워크 플로를 설정하기 만하면 필요한 API 작업이 자동으로 처리됩니다. 다음 섹션에 자세히 설명 된대로 설정할 수있는 스케줄러 워크 플로에는 두 가지 종류가 있습니다.

스케줄러 시작 / 중지

스케줄러를 사용하면 SEP 작업에 대한 활성화 및 비활성화 시간을 설정할 수 있으며 시간을 업데이트 할 수도 있습니다. SEP 스트림의 활성화/비활성화예약에서 자세한 내용을 참조하십시오 .

클립 스케줄러

라이브 작업에서 클립 생성을 예약 한 다음 나중에 일정을 업데이트하거나 변경 사항이있는 경우 클립을 취소 할 수도 있습니다. 자세한 내용은 라이브 스트림에 대한 클립 생성예약에서 참조하십시오 .

관련 주제

  • SEP 스트림의 활성화 / 비활성화 예약
  • 라이브 스트림을위한 클립 생성 예약
  • 라이브 스케줄러 알림
  • 정적 진입 점 (SEP) 작업
  • 라이브 API 참조
  • 관제실 라이브 이벤트 자동 시작 / 중지
  • 관제실 라이브 이벤트 자동 시작 / 중지
