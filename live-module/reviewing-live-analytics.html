    라이브 분석 검토

    이 항목에서는 라이브 모듈에서 제공하는 라이브 분석에 대해 배웁니다.

    라이브 모듈은 라이브 스트리밍 이벤트 또는 채널 실적에 대한 보고서를 제공합니다. 실시간 동시 시청자 및 시청자의 스트림 시간, 지역 및 장치에 대한 집계된 사후 이벤트 데이터를 포함한 정보를 사용할 수 있습니다.

    참고:

    • 뿐동시 시청자실시간으로 제공됩니다. 다른 메트릭은 표준 분석과 유사한 지연으로 수집 및 집계됩니다.
    • 동시 시청자 Brightcove Player 및 SDK와 함께 제공되는 스트림에 대해서만 정확합니다.
    • 이벤트 종료 후 최대 32일 동안 데이터를 사용할 수 있습니다.
    • 실시간 은 1분 간격으로 집계된 세션에 의해 정의됩니다.
    • 반복 이벤트 (정적 진입점) 를 사용하면 최신 활성화 시작 시점부터 분석이 제공됩니다.
    • 현재 분석 모듈은 정확한 “고유 뷰어” 데이터를 가져올 수 없으며 LIVE 모듈 분석 간에 불일치가 있습니다. 라이브 분석 UI를 사용하여 시청자 수를 확인하십시오.
    • 일부 광고 차단기는 실시간 분석 데이터가 제대로 로드되지 않도록 합니다.

      광고 차단기를 비활성화하면 문제가 해결됩니다.

    라이브 분석을 보려면 라이브 모듈을 열고 이벤트 이름을 클릭 한 다음 보고 왼쪽 탐색에서. 분석 대시보드는 다음 섹션으로 구성됩니다.

    • 스냅 사진
    • 장치 별보기
    • 국가 별 조회수

    스냅샷

    스냅샷 섹션이 표시됩니다.

    • 총 조회수 -실시간 스트림 기간 동안의 총 조회수입니다.
    • 고유 한 시청자 -스트림을 시청 한 총 고유 시청자 수. 예를 들어, 동일한 사용자가 한 시간 내에 두 번 개별 실시간 스트림에 연결하면 고유한 뷰어 1개로 계산됩니다.
    • 평균 조회 시간 -주어진 세션에 대한 평균 소비 시간. 예를 들어 사용자 A가 1분의 콘텐츠를 시청하고 사용자 B가 2분의 콘텐츠를 시청하는 경우 평균 보기 시간은 1.5분입니다.
    • 총 시청 시간 -에 표시된 스트림의 지속 시간 동안 소비 된 총 시간시 : 분 : 초 .
    • 광고 시작 -광고 시작 횟수

    스냅샷 아래에 동시 뷰어를 표시하는 그래프가 표시됩니다. 동시 시청자는 위치에 관계없이 지정된 시간에 스트림에 연결된 세션 수를 측정합니다. 예를 들어 10명의 사용자가 라이브 에지에서 라이브 스트림을 보고 있고 5명의 사용자가 DVR 창에서 시청하는 경우 (라이브 후 10분) 분당 동시 시청자는 15명의 사용자가 됩니다.

    장치별 조회 수

    장치 별보기 섹션이 표시됩니다.  기기 별 조회수 그래프 가능한 장치 유형은 다음과 같습니다.

    • 데스크탑  -데스크톱 및 노트북 컴퓨터 포함
    • 태블릿  -iPad, Android 태블릿 등이 포함됩니다.
    • 변하기 쉬운  - 휴대 전화
    • 연결된 TV / 게임 콘솔 / 컴패니언 장치  -스마트 TV, 셋톱 박스 및 게임 콘솔 포함
    • 다른  -위의 어느 것도 매핑되지 않은 사용자 에이전트의 경우

    국가별 조회 수

    국가별 보기 섹션에는 국가별 뷰 수가 포함된 그래프가 표시됩니다.
