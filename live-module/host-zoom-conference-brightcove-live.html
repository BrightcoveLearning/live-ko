소개

Zoom Video Communications는 원격 회의, 재택 근무 및 원격 교육을 위한 클라우드 기반의 P2P 소프트웨어 플랫폼을 제공하는 기술 회사입니다. 더 많은 청중에게 다가 가기 위해 Brightcove Live를 활용하여 다른 사람들이 볼 수 있도록 회의를 스트리밍 할 수 있습니다.

단계

Brightcove Live에 대한 컨퍼런스 또는 웨비나를 주최하려면 다음 단계를 따르십시오.

  1. 컨퍼런스

    Zoom 회의의 경우 다음을 수행하십시오.

    1. 확대/축소 회의설정으로이동하여 미팅 중 (고급섹션으로 스크롤합니다.
    2. 토글 스위치를 클릭하여 라이브 스트리밍 회의허용옵션을 활성화합니다. 이 옵션이 보이지 않으면 Zoom 계정 관리자에게이 기능을 계정에 추가하는 방법에 대해 문의하십시오.

    3. 사용자 지정 라이브 스트리밍 서비스를선택하고 선택적으로 상자에 텍스트 노트를 입력합니다.

    웨비나

    Zoom 웨비나의 경우 다음을 수행하십시오.

    1. Zoom 웨비나 설정으로 이동하여 다음으로 스크롤하십시오. 웨비나 설정에서부분.
    2. 호스트가 웹 세미나를 실시간 스트리밍할 수있도록 허용옵션을 활성화합니다.

    3. 사용자 지정 스트리밍 서비스를선택하고 저장합니다.

  2. 비디오 클라우드 스튜디오에서 미디어 모듈을 사용하여라이브 이벤트를 만듭니다 .

    브로드캐스트를 완료한 후 저장하려면 완료 시 이벤트를 비디오 에셋으로변환확인란을 선택하십시오!

  3. 라이브 이벤트를 생성하면 스트리밍 URL 및 스트리밍 키에 액세스 할 수 있습니다.

    • 스트리밍 URL-스트리밍 엔드 포인트 URL (Brightcove Live Control Room에서 엔드 포인트 URL을 찾을 수 있습니다.
    • 스트리밍 키 - 스트림 이름 (브라이트코브에서는 스트림 이름이 항상“살아있음”)
  4. 웹 사이트 / 인트라넷 페이지에이 라이브 이벤트에 대한 플레이어를 게시하고 포함하고 페이지 URL을 기록해 둡니다.
  5. 컨퍼런스

    확대/축소 컨퍼런스의 경우예약된 모임목록으로 이동하여 스트리밍할 회의를 선택합니다.

    웨비나

    확대/축소웨비나의 경우 웹 세미나 화면으로돌아가 웹 세미나를 예약합니다. 라이브 스트리밍탭을 선택합니다.

  6. 라이브 스트리밍섹션으로 스크롤합니다.
  7. 라이브 스트림 설정구성링크를 선택합니다. 여기에 스트리밍 URL, 스트리밍 키 및 플레이어를 호스팅하는 페이지의 URL을 입력합니다.
  8. 확대/축소 통화에서 MORE버튼을 선택하고 사용자 지정라이브 스트리밍 서비스옵션을 선택합니다. 라이브 스트림이 시작됩니다.
