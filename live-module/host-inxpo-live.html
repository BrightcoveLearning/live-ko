소개

INXPO TV 스타일의 이벤트 경험을 만들기위한 비디오 스트리밍 플랫폼입니다. 더 많은 청중에게 다가 가기 위해 Brightcove Live를 활용하여 이벤트를 스트리밍 할 수 있습니다.

단계

Brightcove Live에 대한 회의를 주최하려면 다음 단계를 따르십시오.

  1. 비디오 클라우드 스튜디오에서라이브 모듈을 사용하여 라이브 이벤트를 만듭니다 .
    브로드캐스트를 완료한 후 저장하려면 완료 시 이벤트를 비디오 에셋으로변환확인란을 선택하십시오!

  2. 라이브 이벤트를 만든 후제어실패널에서 스트리밍 RTMP URL 및 스트리밍 키를 복사합니다.

    • 스트리밍 RTMP URL- 스트리밍 끝점 URL (Brightcove 라이브 관제실에서 엔드 포인트 URL을 찾을 수 있습니다.
    • 스트리밍 키 - 스트림 이름 (브라이트코브에서는 스트림 이름이 항상“살아있음”)
  3. 웹 사이트 / 인트라넷 페이지에이 라이브 이벤트에 대한 플레이어를 게시하고 포함하고 페이지 URL을 기록해 둡니다.

스트리밍 세션 구성

스트리밍 RTMP URL 및 스트리밍 키를 사용하여 INXPO 생산자 도구에서 구성해야합니다.

  1. INXPO 생산자 도구로 이동하고 점 3 개 아이콘을 클릭하여 출력 세부 정보 섹션으로 이동합니다.
  2. 연필 아이콘을 클릭합니다.
  3. Brightcove 라이브 이벤트 자격 증명으로 다음 세부 정보를 입력하십시오.
  4. 제목필드에 브라이트코브 라이브 스트림 이름을 추가합니다.
  5. 인코딩 프로필필드에서 인코더가 압축하고 참석자에게 웹캐스트를 전달하는 방법을 정의하는 기본 인코딩 프로파일을 선택합니다.
  6. RTMP URL필드에 브라이트코브 스트리밍 엔드포인트 URL을 삽입합니다.
  7. 확인버튼을 클릭하면 바로 그 것입니다. 라이브입니다!
