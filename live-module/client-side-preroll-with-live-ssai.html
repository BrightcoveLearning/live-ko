페이지 내용

    라이브 SSAI를 통한 클라이언트측 프리롤 광고

    이 항목에서는 Brightcove Player를 사용하여 SSAI (서버측 광고 삽입) 가 활성화된 라이브 스트림으로 클라이언트측 프리롤 광고를 재생하는 방법에 대해 알아봅니다.

    개요

    브라이트코브 Player를 사용하여 서버 측 광고 삽입 (SSAI를) 통해 라이브 스트림을 제공할 때 클라이언트측 프리롤 광고를 삽입할 수 있습니다. 클라이언트 측 광고의 경우 이 기능은 IMA 광고를 지원합니다.

    이 기능을 사용하려면 다음이 필요합니다.

    • 플러그인 버전 3.5.0+
    • SSAI 플러그인 버전 1.11.0 이상
    • 브라이트코브 플레이어 버전 6.44.0 이상

    주의 사항

    • 이 기능을 사용하는 고객은 동적 배송을 사용해야 합니다.
    • IMA 및 SSAI플러그인만 지원됩니다 (FreeWheel 없음).
    • 현재이 기능은 페이지의 단일 플레이어 만 지원합니다.

    시작하기

    라이브 SSAI 스트림으로 클라이언트측 IMA 프리롤 광고를 재생하려면 다음 단계를 따르십시오.

    1. SSAI가 활성화된 라이브 스트림 만들기
    2. 브라이트코브 플레이어 만들기
    3. Studio를 사용하여 클라이언트 측 프리롤 구현

    SSAI가 활성화된 라이브 스트림 만들기

    라이브 모듈은 서버 측 광고 삽입 (SSAI를) 지원하므로 라이브 스트림 중에 서버 측 광고를 요청하고 표시할 수 있습니다. 실시간 스트림을 만들려면 다음을 참조하십시오.

    • 라이브 모듈에서 서버 측 광고 구현

    브라이트코브 플레이어 만들기

    비디오 클라우드 스튜디오를 사용하여 새로운 브라이트코브 플레이어를 만듭니다. 자세한 내용은빠른 시작: 플레이어 문서 만들기 및 스타일 지정

    브라이트코브 플레이어 버전 6.44.0이상을 사용해야 합니다.

    Studio를 사용하여 클라이언트 측 프리롤 구현

    자동 장애 조치 광고를 위해 플레이어를 구성하는 가장 쉬운 방법은 Video Cloud Studio를 사용하는 것입니다. 광고 구성 및 플레이어를 만든 후에는 다음과 같이 자동 장애 조치 플레이어를 구성 할 준비가 된 것입니다.

    1. 플레이어 모듈을 열고 광고 기능을 추가할 플레이어를 찾습니다.
    2. 플레이어의 속성을 열 플레이어 링크를 선택합니다.
    3. 왼쪽 탐색 메뉴에서광고를선택합니다.
    4. 클라이언트측 (IMA) 사용확인란을 선택합니다.

    5. IMA 광고 태그의 URL을 포함합니다. 이 예에서는 샘플 광고 태그 URL을 사용합니다.

      플레이어 광고 속성에 대한 자세한 내용은 플레이어모듈을 사용하여 플레이어 광고 구성문서를 참조하십시오.

    6. 서버 측 사용 (SSAI를)확인란을 선택합니다.

      계정이 SSAI를 사용하도록 설정되어 있지 않으면 서버 측 옵션이 나타나지 않습니다.
    7. 구성 선택드롭다운 메뉴에서 이 플레이어와 연결할 광고 구성을 선택합니다.

    8. 오버레이를 광고 위에 표시하려면광고 정보 오버레이 사용확인란을 선택합니다. 여기에는 '자세히 알아보기' 및 광고 카운트 다운 오버레이가 포함됩니다.

      오버레이는 여기에서 활성화하고 해당 광고의 광고에 구성하기 전까지는 표시되지 않습니다.
    9. 저장버튼을 선택합니다.

    10. 왼쪽 탐색 메뉴에서JSON 편집기를선택하십시오.

    11. JSON 편집기에서ad_failover: true속성이 표시될 때까지 아래로 스크롤합니다.

    12. ad_failover: true속성을 다음과 같이 바꿉니다. 

      "ima_preroll_with_ssai": true

    13. JSON 편집기는 다음과 비슷해야합니다.

    14. 저장버튼을 선택합니다.

    15. 플레이어를 게시하려면게시 및 포함 변경 내용 > 게시를선택합니다.

    16. 이제 라이브 이벤트를 게시할 준비가 되었습니다. 자세한 내용은라이브 모듈에서 서버 측 광고 구현문서를 참조하십시오.
