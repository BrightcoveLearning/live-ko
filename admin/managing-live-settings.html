라이브 설정 관리

이 항목에서는 Brightcove Live의 설정을 관리하는 방법에 대해 알아 봅니다.

라이브 설정을 사용하면 채우기 슬레이트 및 라이브 자격 증명을 관리 할 수 있습니다.

참고: Brightcove Live에서 사용하는 서버 측 광고 구성은 서버 측 광고 설정 관리의 일부로 구성됩니다. 자세한 내용은 서버 측 광고 설정구성을 참조하십시오 .

채우기 슬레이트 관리

채우기 슬레이트는 실시간 스트림 광고 시간에 공백이있을 때 게재 할 수있는 콘텐츠입니다. 슬레이트는 "곧 돌아 오십시오"라는 메시지 또는 표시하려는 기타 콘텐츠를 제공하는 데 사용할 수 있습니다. 모든 Zencoder 입력 형식이지원됩니다.

참고: 슬레이트 채우기는 Live API를 사용하여 관리 할 수도 있습니다. 자세한 내용은브라이트코브 라이브 API: 서버 측 광고 삽입 (SSAI) .

채우기 슬레이트를 추가하려면 다음 단계를 따르세요.

  1. Studio 헤더에서관리자 > 라이브 설정링크를 클릭합니다.
    참고: ADMIN 메뉴에 액세스하려면 계정 관리자여야 합니다.
  2. + 채우기 슬레이트 추가를 클릭합니다 .
  3. 이름을 입력합니다 .
  4. 슬레이트의소스 URL을입력합니다.
  5. 저장을 클릭합니다 . 슬레이트에 대한 수집 프로세스가 시작됩니다.

채우기 슬레이트를 삭제하려면 삭제 버튼() 채우기 슬레이트 옆에 있습니다.

자격 증명 관리

자격 증명은 Live API 및 Live 모듈 모두에서 사용됩니다. 새 Live Video Cloud 자격 증명을 만드는 경우 먼저 API 인증 자격 증명을 만들어야합니다. 생성 된 클라이언트 ID 및 클라이언트 암호는 Live Video Cloud 자격 증명을 생성 할 때 사용됩니다. FTP, S3 또는 RTMP 자격 증명을 생성 할 때 API 인증 자격 증명이 필요하지 않습니다.

S3, FTP 또는 RTMP 자격 증명을 생성하는 경우 5단계로 건너뜁니다 . Live Video Cloud 자격 증명을 생성하려면 먼저 API 자격 증명을 생성하십시오.

  1. Studio 헤더에서 ADMIN링크를 클릭합니다.
    참고: ADMIN 메뉴에 액세스하려면 계정 관리자여야 합니다.
  2. API 인증링크를 클릭합니다. API 인증 페이지가 열리고 현재 클라이언트 등록이 표시됩니다.
  3. 새 애플리케이션등록을 클릭하여 새 API 자격 증명을 생성합니다. 다음 API를 선택해야합니다.
    참고: API 인증 자격 증명 생성에 대한 자세한 내용은 API 인증 자격 증명관리를 참조하십시오 .
  4. 자격 증명이 생성되면클라이언트 ID및클라이언트 암호값을 저장합니다. Live Video Cloud 자격 증명을 만들 때 필요합니다.
  5. Studio 헤더에서관리자 > 라이브 설정링크를 클릭합니다.
  6. + 자격 증명 추가를 클릭합니다 .
  7. 레이블을 입력합니다 . 신임 정보를 참조하기 위해 내부적으로 사용되는 이름입니다.
  8. 유형 선택 :
    • FTP -VOD 자산을 ftp 위치에 저장하기 위해 사용 (API 사용)
    • RTMP -인코더에서 사용할 RTMP 자격 증명을 만드는 데 사용됩니다.
    • S3 -VOD 자산을 Amazon S3 계정에 저장하는 데 사용 (API 사용)
    • 비디오 클라우드 -VOD 자산을 Video Cloud 계정에 저장하기 위해 사용 (API 사용)
  9. 공개 키/사용자 이름에대해 자격 증명 유형에 따라 다음을 입력합니다.
    • FTP -FTP 사이트의 사용자 이름을 입력하십시오.
    • RTMP -인코더가 인증에 사용할 사용자 이름을 입력하세요.
    • S3 - S3 계정의 사용자 이름을 입력합니다.
    • 비디오 클라우드 - 들어가다클라이언트 ID API 인증 자격 증명에 대해 생성됨
  10. 개인키/암호의 경우 자격 증명 유형에 따라 다음을 입력합니다.
    • FTP - FTP 사이트의 암호를 입력합니다.
    • RTMP -인코더가 인증에 사용할 비밀번호를 입력합니다.
    • S3 - S3 계정의 암호를 입력합니다.
    • 비디오 클라우드 - 들어가다클라이언트 비밀 API 인증 자격 증명에 대해 생성됨
  11. 저장을 클릭합니다 .

자격 증명이 자격 증명 목록에 나타납니다. 자격 증명을 삭제하려면 삭제 버튼() 자격 증명 옆에 있습니다.
