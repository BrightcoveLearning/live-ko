페이지 내용

    사용자 정의 라이브 인제스트 프로파일에 대한 필드

    이 항목에서는 Live의 인제스트 프로파일에 사용되는 필드에 대한 참조를 제공합니다.

    프로파일 필드

    부동산 유형 설명 기본값
    account_id 정수 Video Cloud계정 ID 없음
    name 끈 프로필 이름 - 계정 내에서 고유해야 합니다. 없음
    display_name 끈 Studio 관리 페이지에 표시할 프로파일의 표시 이름 - 와name같을 수 있지만 사용자가 프로파일의 용도를 이해할 수 있도록 설명해야 합니다. 없음
    description 끈 프로파일 설명 없음
    model_version 정수 프로파일이 라이브 모듈에 표시되도록1하려면 이 옵션을 로 설정합니다. 없음
    renditions 배열 변환 객체의 배열 []
    packages 배열 DRM 패키지의 배열 - 현재 DRM은 지원되지 않으므로 항상 빈 배열이어야합니다. []

    변환 필드

    아래 표에서는 요청 본문에서 라이브 인제스트 프로파일을 업데이트하는 데 사용할 수 있는 필드를 보여 줍니다.

    부동산 유형 설명 기본값
    audio_bitrate 정수 오디오 비트레이트 (kbps) 소스 일치
    audio_codec 끈 오디오 코덱 - aac | mp3 형식 또는 프로파일에 의해 결정됨 < - 대부분의aac경우, 최고/td >
    decoder_bitrate_cap 정수 kbps 단위로, 버퍼를 통해 디코더로 공급되는 최대 비트 레이트 없음
    decoder_buffer_size 정수 를 사용할 때 디코더로 공급되는 버퍼의 크기는 kbps로bitrate_cap 없음
    forced_keyframe_rate 번호 X 프레임마다 키프레임을 강제하지만 장면 잘라내기 등에 추가 키프레임을 허용합니다. 없음
    format 끈 비디오 형식 - ts , mp4 , flv 없음
    h264_profile 끈 H.264에는 일반적으로 사용되는 세 가지 프로파일이 있습니다Baseline (최저) Main , 및High . [ 1-5] baseline
    hls_optimized_ts 부울 HLS에 최적화된 시간 세그먼트 - HLS 출력에만 적용 없음
    keyframe_interval 정수 키프레임 사이의 최대 프레임 수 250
    keyframe_rate 흙손 초당 최대 키프레임 수 keyframe_interval중 250명
    label 끈 이미지 변환의 이미지 유형 - poster또는thumbnail 없음
    live_stream 부울 이것이 라이브 비디오 스트림인지 여부 없음
    live_dvr_sliding_window_duration 정수 LiveDVR 전송에 사용할 수 있는 스트림 지속 시간 (초) 300
    max_video_bitrate 정수 킬로 단위, 123 = 123kbps - h264 만 없음
    max_frame_rate 흙손 는 설정하는frame_rate것이 아니라 를 제한합니다. 계정이 60 FPS에 대한 지원에서 활성화되지 않은 경우 30으로 설정되어야 합니다. 없음
    media_type 끈 video또는audio 없음
    video_bitrate 정수 킬로 단위, 123 = 123킬로바이트까지 없음
    video_codec 끈 예: h264, ts 형식 또는 프로파일에 따라 결정
    width 정수 프레임 폭 (픽셀 단위) 없음
    height 정수 프레임 높이 (픽셀) 없음
